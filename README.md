# DOCX to PDF Converter

A web application that allows users to upload `.docx` files, view metadata, and download them as PDFs.

## Features
- Upload `.docx` files
- View file metadata
- Download the converted PDF
- Dockerized for easy deployment

## How to Run
### Using Docker Compose
docker-compose up
